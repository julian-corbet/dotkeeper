Syncthing handles instant file sync. Git handles history. dotkeeper runs both.

Single binary

Syncthing compiled in. No separate install, no daemon, no conflicts.

No servers

Peer-to-peer. No cloud, no accounts, no subscriptions. Your files stay yours.

Git history

Auto-commit, auto-push. Every change versioned. Roll back anytime.

Every platform

Linux, macOS, Windows, BSDs. Native service management on each.

Private

No telemetry, no tracking, no crash reports. Fully offline-capable.

Conflict-free

Staggered git timers per machine. Pushes never collide.

How it works

terminal
$ dotkeeper init --name laptop
[dotkeeper] machine: laptop (slot 0)
[dotkeeper] Syncthing identity generated
[dotkeeper] device ID: KFZZ7TP-GH5OLNL-WNSTHWQ-...

$ dotkeeper add ~/dotfiles ~/projects/webapp
[dotkeeper] added: dotfiles   → ~/dotfiles (git: yes)
[dotkeeper] added: webapp     → ~/projects/webapp (git: yes)

$ dotkeeper status
=== Syncthing ===
  Status: running     Peers: 1

=== Repos ===
  dotfiles  ~/dotfiles          [ok, git, syncthing]
  webapp    ~/projects/webapp   [ok, git, syncthing]

=== Git Backup ===
  Timer: active (daily)
